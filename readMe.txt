本資料夾包含：

1. Alarm.wav
	用於鬧鐘與計時器提醒之鈴聲檔案。

2. Clock
	專案資料夾，包含電子鐘系統之原始碼、標頭檔與資源檔。

3. main.c  
	電子鐘系統之主程式，負責整體流程控制與畫面顯示，
	包含主畫面 UI、時間與日期顯示、時區與手動時間設定，
	並整合鬧鐘與計時器等功能模組。

4. main.h  
	電子鐘系統之主要標頭檔，定義各模組共用之全域變數、
	Console 相關結構，以及數字顯示與時間座標等共用資料。

5. README  
	本專案之說明文件。

6. timer_alarm.c  
	計時器與鬧鐘之原始碼，負責鬧鐘與倒數計時設定、
	響鈴時間判斷，以及 AM / PM 與時間數字之顯示控制。

7. timer_alarm.h  
	計時器與鬧鐘模組之標頭檔，宣告相關功能函式介面。

8. ui_profile.c  
	時制（12/24）、語言切換與背景主題設定之原始碼。

9. ui_profile.h  
	時制、語言與背景主題相關設定之標頭檔。

10. utc_adjust.h  
	時區調整與手動時間設定之模組標頭檔，
	內含時間轉換與走時邏輯（合併原始碼設計）。
